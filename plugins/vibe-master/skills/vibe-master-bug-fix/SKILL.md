# Vibe Master Bug Fix

Use this skill as the Codex-facing equivalent of `/bug-fix`.

## Workflow

1. Reproduce or identify the bug with concrete evidence.
2. Create or locate the bug-fix requirement using existing Vibe Master scripts.
3. Route through `start-work`, `work-on`, review, and test lifecycle steps instead of bypassing manifests.
4. Keep fixes minimal and add regression coverage when practical.

## Report

Return the bug evidence, linked `REQ-ID`, files changed, tests run, and remaining risk.
